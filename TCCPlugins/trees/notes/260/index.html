_DEMOTEWIN — Returns a window handle belonging to the last program started by DEMOTE.

Syntax:
%_DEMOTEWIN

The window handle will be returned in decimal. If the last call to DEMOTE failed to launch the specified command, or if DEMOTE could not find an associated window, _DEMOTEWIN returns 0.

See also: the DEMOTE command and the _DEMOTEPID variable.
