DEMOTE — Run a program without elevation.

Syntax:
DEMOTE /C /K /ALIGN: /CENTER /COLOR: /D: /ENV /INV /MAX /MIN /POS: /Q /SIZE: /WAIT command args

/C

run command in a demoted tcc.exe and close it

/K

run command in a demoted tcc.exe and keep it open

/ALIGN:n,M

align the window relative to a monitor’s edges

/CENTER

center the window on its current monitor

/COLOR:color

set the console window’s startup colors

/D:dir

set the startup directory for the new process

/ENV

create a clean environment block for the new process

/INV

start the new window hidden

/MAX

maximize the new window

/MIN

minimize the new window

/POS:x,y

set the new window’s position

/POS:x,y,w,h

set the new window’s position and size

/Q

quietly

/SIZE:rows,cols

specifies the size of the console screen buffer

/WAIT

wait for the new process to exit

command

the command or program to run

args

any arguments to pass to the demoted program

DEMOTE allows you run a program or command without elevated privileges, from an elevated shell. If you are not running TCC elevated, this command serves no purpose.

If you specify either /C or /K, DEMOTE will launch a non-elevated copy of tcc.exe to run the given command.

If neither /C nor /K is given, the command must name an executable file.

Note that DEMOTE’s options must precede the command on the command line. Any trailing args will be passed on to the demoted command.


/ALIGN:n,M moves the window relative to a monitor. n is required; possible values are:

7: top left

8: top center

9: top right

4: left center

5: center

6: right center

1: bottom left

2: bottom center

3: bottom right

The monitor number M is optional. If is zero or greater than the number of monitors, it defaults to the primary monitor. If you do not specify a monitor number, the window will be aligned on whichever monitor it happens to open on.

/ALIGN cannot be used in combination with /POS:.


/CENTER centers the new window on its current monitor. This option is just short for /ALIGN:5. /CENTER cannot be used in combination with /POS:.


/COLOR:color lets your specify the default colors for the new console window. Note that this option only affects console windows; it has no effect on non-console programs. You can specify your colors either as a hexadecimal value, e.g. /COLOR:0x1f, or as a string, e.g. /COLOR:"BRI WHI ON BLU". The default is boring old white on black.


/POS:rows,cols specifies the size of the console scrollback buffer. Note that Windows treats these values as a suggestion. The system may adjust the dimensions to fit the size of the console window. This option has no effect on non-console programs.


See also: the _DEMOTEPID variable, which returns the process ID of the last program launched by DEMOTE; and the _DEMOTEWIN variable, which returns a window handle belonging to the last program started by DEMOTE.
