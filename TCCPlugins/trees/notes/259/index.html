_DEMOTEPID — Returns the process ID of the last program started by DEMOTE.

Syntax:
%_DEMOTEPID

The process ID will be returned in decimal. If the last call to DEMOTE failed to launch the specified command, _DEMOTEPID returns 0.

See also: the DEMOTE command and the _DEMOTEWIN variable.
