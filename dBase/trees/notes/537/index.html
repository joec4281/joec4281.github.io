The functionality of SQL has been vastly expanded in dBASE IV version 1.1. 


It is now possible to perform a number of dBASE IV commands, using SQL tables, that were previously unavailable. 


This expansion of functionality makes SQL a more attractive feature of dBASE IV version 1.1.


The full description of dot prompt commands that are now supported in version 1.1 when using SQL tables can be found in the section of Getting Started in dBASE IV version 1.1 entitled "What's New in dBASE IV version 1.1."
