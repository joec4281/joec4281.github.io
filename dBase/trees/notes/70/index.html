FUNCTION FDoM

*-----------------------------------------------------------------------

*-- Date........: 01/05/1993

*-- Notes.......: First Day of Month

*-- Written for.: dBASE IV, 1.5

*-- Rev. History: 01/05/1993 -- Original Release

*-- Calls.......: None

*-- Called by...: Any

*-- Usage.......: FDoM(<dArg>)

*-- Example.....: ?FDOM(date())

*-- Returns.....: Date

*-- Parameters..: dArg = a Date argument -- function returns first day

*--                      of the month of this date.

*-----------------------------------------------------------------------


   parameter dArg


RETURN dArg - day( dArg ) + 1
