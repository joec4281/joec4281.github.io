Automatic Date Entry


How can I get a keyboard macro to insert the current date while editing in a screen format?


The following ON KEY LABEL command issued from the dot prompt or in a program prior to a READ or EDIT or BROWSE will set up the F8 function key as a date stamp key. It is sensitive to whether SET CENTURY is ON or OFF and types in the year in appropriate format for either case.


ON KEY LABEL F8 KEYBOARD ;

TRANSFORM(MONTH(DATE()), "@L 99") + ;

TRANSFORM(DAY(DATE()), "@L 99") + ;

IIF(SET("CENTURY") = "ON", STR(YEAR(DATE()), 4), ;

RIGHT(STR(YEAR(DATE()), 4) ,2) )


When you're editing a date field, you simple press F8 and the current system date is entered for you. Make sure your TYPEAHEAD is set to at least 10.
