Q: Is it possible to pass a parameter to a program from the DOS level by entering


   DBASE <prog name> WITH <parameter>


A: With the syntax you provided, the answer is no. Howver, there is a way of parameter passing from DOS into dBASE IV. By making use of DOS environmental variables, a simple DOS batch file, and the GETENV() function in dBASE IV, you can easily pass parameters to a dBASE III PLUS or dBASE IV program.


Create a batch file called DBPROG (a name used for this example but can be any valid DOS filename you choose) that contains the following statements:


@ECHO OFF

SET VAR1=%1

DBASE <prog name> /T

SET VAR1=

CLS


Include the following line in your dBASE program:


mvar1 = GETENV("VAR1")


Execute the batch file by typing the name of the file followed by the parameters you wish to pass at the DOS level:


DBPROG <parameter>


By using the replaceable parameters capabilities of DOS, one can pass multiple parameters (up to nine) to a program.
