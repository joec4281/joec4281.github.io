Here's a simple little program/procedure that toggles between Display and Edit modes (in other words, toggle between @ SAYs and @ GETs).


Setting Flag = .T. will assign the initial mode to Display only. When Flag = .F. the screen will be set to Edit mode allowing changes to the given fields. By adding a parameter statement to the code below, you could pass the true/false value in a called procedure to allow your Display programs and Edit programs to use the same screen definitions. In that case, remove the "ON KEY LABEL F5 DO Flip" command if you want to disable the ability to toggle between display and edit modes.


* SAYGET.PRG

Flag = .T.

mFld1 = Field_1

mFld2 = Field_2


DO WHILE LASTKEY() <> 27

  ON KEY LABEL F5 DO Flip

  mSayGet = IIF(Flag,"SAY","GET")

  @ 10, 10 &mSayGet mFld1

  @ 12, 10 &mSayGet mFld2

  READ

ENDDO

RETURN

*

PROCEDURE Flip

  Flag = IIF(Flag, .F., .T.)

  KEYBOARD CHR(29)

RETURN
