One of the enhanced features of Clipper 5.0 is its array handling. You can now initialize an array with an assignment statement such as;


aNumber := {100,200,"Three Hundred",400}


Not only is this statement a quick way to initialize arrays, it also can be used to parse a comma-separated list of values. Let's suppose that you have read a comma-separated list of values from a data file into the variable LineRead, where


LineRead = '100,200,"Three Hundred",400'


You might then want a function that takes this string as input and returns an array with four values. Writing such a function would normally involve using the AT() function within a loop to locate each comma, while placing each substring into an array element.


* PARSE.PRG

*--- Example comma-separated list

LineRead = '100,200,"Three Hundred",400'

*--- Parse list, placing each element in an array

myArray = GetParmList(LineRead)


*--- Print array values

for x=1 to len(myArray)

  ? myArray[x]

next x

return


Function GetParmList(plist)

return &('{'+plist+'}')
